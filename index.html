Conditionals

What are Conditionals used for?

To control the flow of code based on if a specific condition is true or false
Comparison Operators
Compares two vales with each other
  • > greater than
  • < less than
  • >= greather than or equal to
  • <= less than or equal to
Logical Operators
&& checks if the left AND right side of the comparison is true
  • true&&true = true
  • true&&false = false
  • false&& true = false
  • false&&false = false
Logical Operators pt2
|| checks if the left OR right side of the comparison is true
  • true||true = true
  • true||false = true
  • false||true = true
  • false||false = false
Falsy Values
A falsy value is considered as `false` when convered to a boolean
  • Undefined
  • Null
  • NaN
  • 0
  • ""
  • false
Truthy Values
A truthy value is considered as `true` when convered to a boolean
  • ANY number
  • ANY strings
  • {}
  • []
